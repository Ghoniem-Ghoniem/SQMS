Oracle DBA with GoldenGate - Job Description

Company: I-SOFT

Location: Dubai

Employment Type: Full-time

Industry: Information Technology & Services / Enterprise Solutions

About the Role:

We’re looking for an experienced Oracle Database Administrator (DBA) with deep knowledge of Oracle GoldenGate and Oracle WebLogic to join our high-performing data infrastructure team. This role ensures the stability, performance, and replication of mission-critical databases across multiple environments.

What You’ll Do:

·        Manage and maintain Oracle databases (12c/19c/21c) in production and non-prod.

·        Configure and operate Oracle GoldenGate for real-time data replication and bidirectional.

·        Execute performance tuning, backup/recovery (RMAN), patching, and upgrades.

·        Implement and support RAC, ASM, Data Guard, and GoldenGate for HA/DR.

·        Manage and maintain Oracle WebLogic, Oracle OHS in terms of the installation, configuration, application deployment, monitoring and security implementation.

·        Collaborate with developers & DevOps to optimize database integration and automation.

·        Proactively monitor database health and resolve issues (OEM, scripts).

·        Ensure database security and compliance with internal standards.

 

What You Bring:

·        5+ years of Oracle DBA experience.

·        Solid experience with Oracle GoldenGate for real-time data replication and high availability.

·        Strong experience in Oracle WebLogic Server installation, configuration, and administration.

·        Solid understanding of Oracle RAC, ASM, Data Guard, and RMAN.

·        Solid knowledge of SQL, PL/SQL, Unix/Linux shell scripting.

·        Experience with Oracle Enterprise Manager (OEM) for monitoring and administration.

·        Familiarity with replication across mixed environments (optional).

Bonus Points:

·        Oracle Certified (OCP, OGG, WebLogic) are a plus.

·        Experience in cloud environments (OCI, AWS RDS, Azure).

·        Understanding of microservices architecture and CI/CD pipelines.

Soft Skills:

·        Strong troubleshooting and communication skills.

·        Comfortable working in a fast-paced, agile environment.
