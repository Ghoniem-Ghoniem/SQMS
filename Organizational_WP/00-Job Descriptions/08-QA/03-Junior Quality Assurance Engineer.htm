Job description

Job title: Junior Quality Engineer

 

Department: Web Development

 

Reports to: Higher level (Senior/Junior)

 

 

Job code: QC-JN

 

Job type: full time.

 

 

(SUMMARY) Basic function and scope:

 

This position is responsible for executing software quality plans and helping to ensure that the project is completed on time with the accepted level of quality. This position will also be responsible for executing and follow-up software testing activitie.

 

Main tasks and responsibilities:

1.       Responsible for conducting and coordinating software quality activities throughout the project.

2.       Share in daily stand ups to ensuring sprints are completed on time and removing impediments.

3.       Responsible for quality activities by actively writing testing test cases and issues.

4.       Responsible for executing test cases and reporting issues and follow-up till closure.

5.       Ability to understand analysis and architecture models to build test cases upon.

 

Required skills and qualifications:

  1. Education: B.SC. Of computer science or equivalent.
  2. Experience: 1+ years.

3.       Having basic experience testing types.

4.       Known of configuration management environments.

5.       Excellent English reading and writing skills

6.       Ability to learn new technologies.

7.       Must have excellent problem solving skills

8.       Overcoming technical challenges.

9.       UML knowledge and practice

10.   Testing for both desktop and web applications, mobile application is +

11.   Deployment basics for desktop and web solutions, and databases

12.   SQL Server basics

13.   Ability to prepare testing environment.

14.   Agile methodology concepts and practice
